## Getting Started

### Development

```
git clone https://github.com/vctrblck/whispers.git
cd whispers
npm install
```

### Running the server

1. Run a local HTTP server in this project directory.
2. Open a browser.
3. Request `<server-address>:<port-number>/index.html` from browser.
